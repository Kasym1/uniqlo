2. Базовая структура
    2.1 Создать интерфейс Shop. Прописать методы getItems(), getTotalPage()
    2.2 Создать базовый класс Shop реализующий интерфейс Shop.
    2.3 Создать классы магазинов AmazonShop, UniqloShop, RakutenShop. Они наследуют класс Shop. У классов наследников должны быть свойства url, name, totalPage.
3. Создать переменную $userShop, который проверяется на совпадение магазинов, которые есть и при совпадении подставляет реализацию нужного магазина.
4. У класса магазина обращаются к методу getItems(), который возвращает массив объектов Product

    4.2 метод getTotalPage() возвращает свойство totalPage
